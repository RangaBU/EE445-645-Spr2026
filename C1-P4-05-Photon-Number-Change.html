Change in Photon Distribution Function

Change Δf as photons travel:
Δf = f(r+ΔξΩ, ν, Ω, t+Δt) - f(r, ν, Ω, t)
Photons move with time:
r(t) depends on t Position changes: r → r + ΔξΩ Time advances: t → t + Δt Distance: Δξ = c·Δt
Chain rule needed:
Δf/Δt = ∂f/∂t + (∂f/∂x)(dx/dt) + (∂f/∂y)(dy/dt) + (∂f/∂z)(dz/dt) = ∂f/∂t + (∂f/∂x)(cΩₓ) + (∂f/∂y)(cΩᵧ) + (∂f/∂z)(cΩᵧ)
Final form:
Δf/Δt = (1/c·∂f/∂t + ∂f/∂x·Ωₓ + ∂f/∂y·Ωᵧ + ∂f/∂z·Ωᵧ)·c

Key insight: Position r depends on time t because photons travel. Chain rule accounts for both explicit time dependence (∂f/∂t) and implicit time dependence through position (∂f/∂x·dx/dt, etc.).
